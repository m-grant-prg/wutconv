NAME
wutconv - converts one or more text files to Windows or Unix line endings.
SYNOPSIS
wutconv [OPTION] TextFileName TextFileName1 ...
DESCRIPTION
Converts the line endings in one or more text files to Windows or Unix format, i.e. LFCR or LF.
OPTIONS
-h, --help	Displays usage information.
-u, --unix	Convert to Unix line endings.
-V, --version	Displays version information.
-w, --windows	Convert to Windows line endings.
EXIT STATUS
0
if OK,

1
if minor problems,

64
error in command line arguments,

66
file access error,

67
trap received,

77
permissions error.
